This is a java program that takes the input file name given as argument. 
Flag --verbose is optional.

It will operate the First Come First Served, Round Robin, Last Come First Served, and Preemptive Shortest Job First algorithms.

To compile: javac Scheduling.java SchedulingTools.java Process.java
To run: java Scheduling <input-filename>
(or) 	java Scheduling --verbose <input-filename>

This program prints output to the Console.
